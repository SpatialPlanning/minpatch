# MinPatch for R: Post-processing prioritizr solutions to ensure minimum patch sizes

This package provides functions to post-process conservation planning
solutions from prioritizr to ensure all protected areas meet
user-defined minimum size thresholds, following the methodology
described in Smith et al. (2010).
