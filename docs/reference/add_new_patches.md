# Add new patches to meet conservation targets

Stage 2 of MinPatch: Add new patches using BestPatch algorithm

## Usage

``` r
add_new_patches(minpatch_data, verbose = TRUE)
```

## Arguments

- minpatch_data:

  List containing all MinPatch data structures (including prioritizr
  objects)

- verbose:

  Logical, whether to print progress

## Value

Updated minpatch_data with new patches added
