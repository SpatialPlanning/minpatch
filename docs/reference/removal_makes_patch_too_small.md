# Check if removing unit would make its patch too small

Check if removing unit would make its patch too small

## Usage

``` r
removal_makes_patch_too_small(unit_id, minpatch_data)
```

## Arguments

- unit_id:

  ID of unit to potentially remove

- minpatch_data:

  List containing all MinPatch data structures

## Value

Logical indicating if removal would make patch too small
