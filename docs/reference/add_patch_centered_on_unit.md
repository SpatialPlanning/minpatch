# Add patch centered on specified planning unit

Add patch centered on specified planning unit

## Usage

``` r
add_patch_centered_on_unit(minpatch_data, center_unit_id)
```

## Arguments

- minpatch_data:

  List containing all MinPatch data structures

- center_unit_id:

  ID of the unit to center the patch on

## Value

Updated unit_dict with new patch added
