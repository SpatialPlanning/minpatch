# Check if removing unit would split patch into non-viable pieces

Check if removing unit would split patch into non-viable pieces

## Usage

``` r
removal_splits_patch_nonviably(unit_id, minpatch_data)
```

## Arguments

- unit_id:

  ID of unit to potentially remove

- minpatch_data:

  List containing all MinPatch data structures

## Value

Logical indicating if removal would create non-viable patches
